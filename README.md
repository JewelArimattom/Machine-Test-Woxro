# Machine-Test-Woxro

A simple user management website built with Next.js App Router and React state.

## Features

- Add users with name and age
- Default status is Active for new users
- Toggle status between Active and Inactive
- Edit existing user details and save updates
- Data persistence in browser localStorage

## Tech Stack

- Next.js (App Router)
- React
- TypeScript
- Local state with useState

## Run Locally

1. Install dependencies:

```bash
npm install
```

2. Start development server:

```bash
npm run dev
```

3. Open in browser:

```text
http://localhost:3000
```
